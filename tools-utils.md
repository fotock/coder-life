# 善事之利器

#### Visual Studio Code (Mac)
首选的文本编辑器。可安装的插件非常多。语法、提示、项目、版本控制等等非常丰富。也就是我敲这些文字正在使用的工具。评测说比 Sublime 好用.

#### Vim
服务器端逃不掉的工具。
[>> 常用配置](tools/vim.md)

#### 在线IP地址(经纬度)转换
网站很多。但能比较是否准确的不多。这个网站提供的是比较全的:
https://www.iplocation.net
